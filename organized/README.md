# Projet académique visant à nous apprendre l'utilisation et la manipulation de listes chainées et des algorithmes de tri .

**Compilation** : Makefile via make  
**Éxécution** : ./organized pour rentrer dans le shell  
**Commande** : **add** WIRE/ACTUATOR/DEVICE/SENSOR/PROCESSOR + nom : ajouter un objet (TYPE + nom)  
               **del ID** : supprimer un objet grace à son ID  
               **disp** : afficher la liste chainée  
               **sort** : trier la liste par nom (NAME), type (TYPE) ou ID (ID). Présence d'un flag **-r** pour trier en sens inverse (Ex: sort ID -r)  
               **exit** : sortir du shell
