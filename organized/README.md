# Projet académique visant à nous apprendre l'utilisation et la manipulation de listes chainées et des algorithmes de tri .

**Compilation** : Makefile via make\n
**Éxécution** : ./organized pour rentrer dans le shell\n
**Commande** : **add** WIRE/ACTUATOR/DEVICE/SENSOR/PROCESSOR + nom : ajouter un objet (TYPE + nom)\n
               **del** ID : supprimer un objet grace à son ID\n
               **disp** : afficher la liste chainée\n
               **sort** : trier la liste par nom (NAME), type (TYPE) ou ID (ID). Présence d'un flag **-r** pour trier en sens inverse (Ex: sort ID -r)\n
               **exit** : sortir du shell\n
